# 按键

| ID                  | 默认     | 描述             |
| ------------------- | -------- | ---------------- |
| `open_config_menu`  |          | 打开配置菜单     |
| `toggle_mod_enable` |          | 切换模组是否启用 |
| `toggle_side`       | CapsLock | 切换相机到另一侧 |
| `adjust_position`   | Z        | 按住调节相机位置 |
| `force_aiming`      |          | 按住强制瞄准     |
| `toggle_aiming`     |          | 短按切换瞄准状态 |
| `toggle_pitch_lock` |          | 切换俯仰角锁定   |
